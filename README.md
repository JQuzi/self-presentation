# Self Presentation

Одностраничное портфолио AI Automation Specialist для публикации на GitHub Pages.

Опубликованная версия:
https://jquzi.github.io/self-presentation/

## Локальный запуск

Откройте `index.html` в браузере или запустите простой локальный сервер:

```powershell
python -m http.server 8000
```

После этого сайт будет доступен по адресу `http://localhost:8000`.

## Публикация

Сайт публикуется через GitHub Pages из ветки `main`, папка `/ (root)`.
